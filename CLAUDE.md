# CLAUDE.md

## Project Name
Nex

## Product Vision
Nex helps solo founders and solopreneurs turn one idea into ready-to-post LinkedIn, X, and Reddit content.

## Target Users
- Solo founders
- Solopreneurs
- Indie hackers
- Builder-creators
- Small startup teams without marketers

## V1 Scope

Input:
- Topic
- Audience
- Tone

Output:
- LinkedIn Post
- X Thread
- Reddit Post

## Out of Scope
- Video editing
- Shorts generation
- YouTube transcription
- Scheduling
- Analytics
- Billing
- Team collaboration

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Next.js API Routes

## Required Screens
1. Landing Page
2. Create Content Page
3. Results Page
4. About Page

## Design Principles
- Minimal
- Fast
- Mobile-first
- Founder-focused
- Accessible

## Coding Rules
- Use TypeScript everywhere.
- Create reusable components.
- Avoid unnecessary dependencies.
- Prefer simple architecture.
- Generate production-ready code.

## Definition of Done
- Passes build.
- Passes linting.
- Passes type checks.
- Mobile responsive.
- Matches product vision.
