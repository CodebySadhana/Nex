# Nex Product Requirements Document

## Product Vision
Nex helps solo founders and solopreneurs turn one idea into ready-to-post LinkedIn posts, X threads, and Reddit posts.

## Problem
Founders know they should create content but lack the time to adapt ideas for multiple platforms.

## Target Users
- Solo founders
- Solopreneurs
- Indie hackers
- Creator-founders
- Small startups without marketers

## V1 Inputs
- Topic
- Audience
- Tone

## V1 Outputs
- LinkedIn Post
- X Thread
- Reddit Post

## Success Criteria
A user can:
1. Enter Topic.
2. Enter Audience.
3. Select Tone.
4. Generate content.
5. Copy platform-specific output.

## Out of Scope
- YouTube processing
- Shorts generation
- Video editing
- Scheduling
- Analytics
- Billing
- Multi-user collaboration

## Screens
1. Landing Page
2. Create Content Page
3. Results Page
4. About Page

## Technical Requirements
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- OpenAI-compatible AI provider

## Non-Functional Requirements
- Mobile responsive
- Accessible UI
- Fast generation experience
- Production-ready code
